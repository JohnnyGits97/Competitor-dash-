How to Use This Dashboard

⚠️ This is a static frontend dashboard. The Python scraper must be run separately on your machine to collect live pricing data.
Step 1 — Run the Scraper
pip install requests beautifulsoup4 playwright pandas rich
playwright install chromium
python construction_price_scraper.py
Step 2 — Import the Output

The scraper saves files to an output/ folder. Import the raw_products_*.csv or price_comparison_*.csv file using the Import Data button.

Step 3 — Analyze

Use the Overview and Price Comparison tabs to see benchmark comparisons, "vs Biiibo" deltas, and price leaders for each product category.
